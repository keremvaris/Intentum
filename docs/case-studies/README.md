# Case studies

This folder contains numeric case studies that demonstrate Intentum's effectiveness in specific domains. **Case study documents are in English only;** they are not duplicated under `docs/en` or `docs/tr`.

- [Greenwashing detection metrics](greenwashing-metrics.md) — labeled dataset, transformation rule, accuracy/F1 summary. **Three evaluation paths:** (1) synthetic: `GreenwashingCaseStudy_ComputesAccuracyAndF1` (19 examples); (2) public HTML: run `./scripts/download-greenwashing-sources.sh`, then `GreenwashingCaseStudy_OnDownloadedHtml_ComputesAccuracyAndF1` (ClientEarth); (3) Mendeley Excel: unpack dataset to [downloaded/DataGreenwash/](downloaded/DataGreenwash/), then `GreenwashingCaseStudy_OnMendeleyExcel_ComputesAccuracyAndF1`. Sources: [greenwashing-sources.md](greenwashing-sources.md), [greenwashing-labeled-sources.csv](greenwashing-labeled-sources.csv) (53 rows).
- [Cross-LLM consistency](cross-llm-consistency.md) — optional: how to compare confidence/intent across embedding providers (OpenAI, Azure, Gemini, etc.).
- [Load test: infer endpoint](load-test-infer.md) — k6 script and result summary for Sample.Web `POST /api/intent/infer`.
- [Benchmark results](benchmark-results.md) — BenchmarkDotNet results (ToVector, Infer, PolicyEngine). Generated by `./scripts/run-benchmarks.sh`.
